# Support

## Where to get help

### Discussions

For questions, ideas, and general conversation, use
[GitHub Discussions](https://github.com/kaif-ai-engineer/forge-kaif/discussions).
This is the best place for:

- How-to questions
- Best practices
- Showing off what you built
- Feature brainstorming

### Bug Reports

If you have found a bug, please open a
[bug report](https://github.com/kaif-ai-engineer/forge-kaif/issues/new?labels=bug&template=bug_report.md).
Include steps to reproduce, expected behavior, and your environment.

### Feature Requests

To propose a new feature, open a
[feature request](https://github.com/kaif-ai-engineer/forge-kaif/issues/new?labels=enhancement&template=feature_request.md).
For large or controversial changes, start with a
[Discussion](https://github.com/kaif-ai-engineer/forge-kaif/discussions) first.

### Security Vulnerabilities

If you discover a security vulnerability, please **do not** open a public
issue. Instead, follow the responsible disclosure process in
[SECURITY.md](SECURITY.md).

### Documentation

Full documentation is available at [ForgeKaif](https://kaif-ai-engineer.github.io/forge-kaif/).
If you find gaps or errors, open a
[documentation issue](https://github.com/kaif-ai-engineer/forge-kaif/issues/new?labels=documentation&template=documentation.md).

## Community

- [Discord](https://discord.gg/forge) — real-time chat

## Before opening an issue

1. Check the [documentation](https://kaif-ai-engineer.github.io/forge-kaif/).
2. Search existing [issues](https://github.com/kaif-ai-engineer/forge-kaif/issues)
   and [discussions](https://github.com/kaif-ai-engineer/forge-kaif/discussions).
3. Make sure you are on the latest version.
